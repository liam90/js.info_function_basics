JS Function Basics

Function Decloration

The function keyword goes first, then goes the name of the function, then a list of parameters between the parentheses (comma-separated, empty in the example above, we’ll see examples later) and finally the code of the function, also named “the function body”, between curly braces.

function showmMessage() { alert("Hello World"); }

function nameOfFunction(parameter1, parameter2, paraemter3, ect....) { ... function body ... }

calling the fuunction to run it
nameOfFunction();

Local Variables

A local variable is created and only usable or scoped within the function it was created in.

function localVar() { let local = 'Hello World';
console.log(local); }

localVar();

Outer Variables

A variable that can be used and even changed by any function within the program.

let userName = 'John';

function greetUser () { let message = `Hello ${userName}`;
console.log(message); }

greetUser();

Parameters

We can pass arbitrary data to functions using parameters. In the example below, the function has two parameters: from and text.

function parameterFunc(from, text) { console.log(`${from} : ${text}`); }

parameterFunc('Liam','Success');

Returning a Value

A function can return a value back into the calling code as the result. The directive return can be in any place of the function. When the execution reaches it, the function stops, and the value is returned to the calling code

function addTwoNums(a, b) { return a + b; }

let result = addTwoNums(3, 3);

console.log(result);
